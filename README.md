**AKIRA** is a futuristic, anime-inspired AI companion featuring voice interaction, emotion detection, and a stunning cinematic interface. Built with speed and aesthetics in mind, AKIRA behaves like a real-time assistant with a personality.

---

## ✨ Key Features

- 🎭 **Emotion Detection**: AKIRA senses your mood (Happy, Sad, Angry, Curious, Stressed) and changes her holographic ring colors and response tone to match.
- 🎙️ **Voice Interaction**: High-speed speech-to-text (Google) and natural-sounding text-to-speech (Edge-TTS).
- 📝 **Smart Memory**: Remembers your name, facts about you, and can summarize your recent conversations.
- ⚙️ **Customizable Sensitivity**: Adjustable microphone sensitivity levels for any environment.
- ⚡ **Quick Actions**: One-click shortcuts for Music, Weather, News, and Timers.
- 🌌 **Cinematic UI**: A premium dark-mode interface with glassmorphism, glowing accents, and animated ring effects.

## 🛠️ Setup & Installation

### Prerequisites
- Python 3.9 or higher
- [Groq API Key](https://console.groq.com/keys) (Add to `.env`)

### Fast Install
1. **Clone the Repo:**
   ```bash
   git clone https://github.com/yourusername/akira-companion.git
   cd akira-companion
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment:**
   Create a `.env` file in the root directory:
   ```env
   GROQ_API_KEY=your_actual_key_here
   ```

4. **Launch AKIRA:**
   Double-click `START_AKIRA.bat` or run:
   ```bash
   python gui_companion.py
   ```
