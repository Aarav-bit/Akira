**AKIRA** is a high-end, futuristic AI companion designed to bridge the gap between utility and cinematic experience. Inspired by cyberpunk aesthetics and advanced companion systems, AKIRA doesn't just process commands—she responds with personality, visual emotion, and intelligence.

## 🌌 The Vision
Unlike traditional text-based bots, AKIRA is built as a **Desktop Entity**. She features a stunning holographic interface that reacts in real-time to your voice and emotional state.

---

## ✨ Standout Features

### 🎭 Neural Emotion Detection
AKIRA senses the tone of your conversation. Whether you're stressed, happy, or curious, her holographic ring shifts its glow and animation patterns to match your mood.

### 🎙️ Advanced Voice Intelligence
- **Real-time Interaction**: Near-zero latency response using Groq's Llama-3 architecture.
- **Natural Voice**: Leveraging Edge-TTS for human-like cadence and emotion.
- **Smart Wake-Word**: Optimized for seamless voice activation.

### 🧠 Persistent Smart Memory
AKIRA builds a relationship with you. She remembers your preferences, summarizes past discussions, and maintains a long-term context of your interactions.

### 🛠️ System & Web Integration
- **Web Intelligence**: Real-time web searching for news, weather, and facts.
- **Media Control**: Voice-activated playback for Spotify and system media.
- **Device Mastery**: Control brightness, volume, and system settings with natural language.

## 🛠️ Quick Start

### 1. Prerequisites
- Python 3.9+
- A [Groq API Key](https://console.groq.com/keys)

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/akira-companion.git
cd akira-companion

# Install dependencies
pip install -r requirements.txt
```

### 3. Configuration
Create a `.env` file in the root directory:
```env
GROQ_API_KEY=your_key_here
```

### 4. Launch
Launch via the batch file for the best experience:
```bash
START_AKIRA.bat
```
